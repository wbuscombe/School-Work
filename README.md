# School Work

A collection of programming projects and assignments from various computer science courses.

## Courses

### Algorithms
- **Topological Sort** - Graph algorithm implementation with cycle detection using depth-first search

### CS-II-Java
- **Lab-01: Pie Chart** - Graphics/visualization with random colored slices
- **Lab-05: Chess Pieces** - GUI demonstrating inheritance and polymorphism with chess piece movement
- **Lab-06: Sorted Gradebook** - Selection sort implementation with student records
- **Lab-08: Monetary Value** - Currency calculation with exception handling
- **Lab-09: File I/O** - Random circle generation with custom exceptions
- **Lab-10: Sierpinski Triangle** - Recursive fractal drawing with iteration controls

### Data-Structures-Java
- **Homework-4: Josephus Problem** - Classic elimination problem solver
- **Homework-6: Union-Find** - Weighted quick union and randomized queue implementations
- **Homework-7: Graphical Sorting** - Visual insertion and selection sort demonstrations
- **Homework-8: Collinear Points** - Computational geometry with brute force and optimized algorithms

### Intermediate C Programming
- **Projects 01-05** - Basic C programs (loan calculator, investment tools)
- **Project03_01: Minesweeper** - SDL2 game with timer, sound, and animations
  - [Pre-built binaries](Intermediate%20C%20Programming/Project03_01/README.md) available for Linux, macOS, and Windows
- **Project06_01: IMDB Database** - SQLite database system with C client and Java builders
