# School Work

A collection of programming projects and assignments from various computer science courses.

## Courses

### Algorithms
- **[Topological-Sort](Algorithms/Topological-Sort/)** - Graph algorithm with cycle detection using depth-first search

### CS-II-Java
- **[Lab-01: Pie Chart](CS-II-Java/Lab-01/)** - Graphics/visualization with random colored slices
- **[Lab-05: Chess Pieces](CS-II-Java/Lab-05/)** - GUI demonstrating inheritance and polymorphism
- **[Lab-06: Sorted Gradebook](CS-II-Java/Lab-06/)** - Selection sort with student records
- **[Lab-08: Monetary Value](CS-II-Java/Lab-08/)** - Currency calculation with exception handling
- **[Lab-09: File I/O](CS-II-Java/Lab-09/)** - Random circle generation with custom exceptions
- **[Lab-10: Sierpinski Triangle](CS-II-Java/Lab-10/)** - Recursive fractal drawing

### Data-Structures-Java
- **[Homework-4: Josephus Problem](Data-Structures-Java/Homework-4/)** - Classic elimination problem solver
- **[Homework-6: Union-Find](Data-Structures-Java/Homework-6/)** - Weighted quick union and randomized queue
- **[Homework-7: Graphical Sorting](Data-Structures-Java/Homework-7/)** - Visual sorting demonstrations
- **[Homework-8: Collinear Points](Data-Structures-Java/Homework-8/)** - Computational geometry algorithms

### Intermediate-C-Programming
- **[Loan-Balance-Calculator](Intermediate-C-Programming/Loan-Balance-Calculator/)** - Calculates remaining loan balance after payments
- **[Magic-Square](Intermediate-C-Programming/Magic-Square/)** - 4x4 matrix with row/column/diagonal sums
- **[GCD-Calculator](Intermediate-C-Programming/GCD-Calculator/)** - Greatest Common Divisor calculator
- **[Eulers-Number](Intermediate-C-Programming/Eulers-Number/)** - Approximates Euler's number (e) via infinite series
- **[Fraction-Calculator](Intermediate-C-Programming/Fraction-Calculator/)** - Four-function calculator with fraction support
- **[Minesweeper](Intermediate-C-Programming/Minesweeper/)** - SDL2 game with timer, sound, and animations
- **[IMDB-Database](Intermediate-C-Programming/IMDB-Database/)** - SQLite query tool for movie data
